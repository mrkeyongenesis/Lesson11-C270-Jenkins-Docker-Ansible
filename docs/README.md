# Lesson Docs Index

These files are the student-facing, command-focused lesson guides for the repo's RHEL 9 VM lab.

Use these files in order:

- `docs/LESSON10_PARTED_RHEL9.md` — RHEL 9 partitioning with `parted`, GPT, EFI, and `xfs`.
- `docs/LESSON11_JENKINS_PIPELINE_RHEL9.md` — Jenkins pipeline setup and running the repo `Jenkinsfile`.
- `docs/LESSON12_ANSIBLE_RHEL9.md` — Ansible deployment for the two-tier FastAPI + Streamlit app.

These lesson files are written for beginner students and focus on exact commands, expected results, and simple verification steps.

> Note: The repo also contains longer conceptual companion guides in `docs/JENKINS_PIPELINE.md` and `docs/ANSIBLE_DEPLOY.md`.
